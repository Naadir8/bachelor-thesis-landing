# Intelligent System for Test Scenario Grouping and Evaluation
This project represents the web-based part of a diploma thesis focused on developing an intelligent system for grouping and evaluating test scenarios in the automated software testing process. The website serves as a structured presentation of the research, including its objectives, methodology, and expected results.

# Goal
To design and implement a web interface that presents the diploma project, ensuring clear communication of the research topic while supporting accessibility, SEO optimization, and bilingual localization (Ukrainian / English).

# Main tasks solved in this version
- Initialize the web project structure
- Implement the base HTML layout
- Ensure semantic markup and accessibility (ARIA attributes, alt texts)
- Add SEO optimization (meta tags, Open Graph, schema.org)
- Implement full content according to diploma requirements
- Support bilingual interface (UA / EN language switch)
- Develop responsive design for multiple devices
- Optimize performance (images, structure, loading)

# Stack and requirements
- HTML5 (semantic markup)
- CSS3 (styling and responsiveness)
- JavaScript (language switching logic)
- Git + GitHub (GitHub Flow strategy)

# Key components
- index.html — main webpage
- styles.css — styling and responsive design
- script.js — language switch logic (UA/EN)
- robots.txt — search engine indexing rules
- sitemap.xml — website structure for crawlers
- assets/ — images, icons, favicon
